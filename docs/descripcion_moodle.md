# Descripcion lista para Moodle

## Nombre de la actividad
Semana 13: Artefacto del entorno - configuracion, script y depuracion de errores

## Tipo de actividad
Tarea / Assignment

## Modalidad
Individual o grupal, segun avance.

## Consigna
Desarrolla un artefacto tecnico ejecutable a partir del proyecto base entregado en ZIP. El proyecto contiene archivos CSV con miles de registros y un script principal con varios errores intencionales. Debes abrir el proyecto en Visual Studio Code, ejecutar los scripts, identificar los errores, corregirlos, generar un reporte y documentar el proceso.

## Evidencia de entrega
Sube un archivo ZIP con el proyecto corregido. El ZIP debe contener el codigo funcional, el reporte generado, la bitacora de errores, evidencias de pruebas y la matriz de comparacion tecnica del entorno.

## Criterios tecnicos
- Producto ejecutable.
- Organizacion de archivos.
- Evidencia de pruebas.
- Coherencia con el problema contextual.
- Correccion documentada de errores.
- Analisis de funcionalidad, compatibilidad y costo.
